# KOSPI AI 모니터링 시스템 스냅샷 및 차세대 발전 계획 (Self-Evolution)

본 문서는 재부팅 및 다음 세션 가동 시 현재까지의 작업 상황과 설계된 미래 아키텍처를 AI가 즉시 인지하도록 기록되었습니다.

## 1. 현재 시스템 요약 (Done)
- **무료 서버리스 아키텍처**: GitHub Actions 기반 0원 운영 중.
- **휴장일 자동 스킵**: `holidays`를 통한 주말/공휴일 자동 감지 및 강제 종료 로직 탑재.
- **비상 백업 모드**: 구글 AI(Gemini) Quota 초과 시 에러 메시지 없이 "순수 데이터 전문"만 깔끔하게 전송하는 로직 완성.
- **한국인 패치**: 모든 기술적 분석 영문 명칭을 직관적인 한국어로 번역 완료.

## 2. 차세대 목표: 자가 진화(Self-Evolution) 엔진
유저의 개입 없이 스스로 공부하고 전략을 수정하는 완전 자율 투자 로직을 설계 중입니다.

### 설계 방향
1. **유전자 추출(Parameterization)**: `analyzer.py`에 고정된 상수(RSI 14, 이평선 200 등)를 `strategy_config.json`으로 분리하여 가변형 구조로 변경.
2. **자가 학습 엔진(`optimizer.py`)**: 주말마다 지난 일주일간의 성적을 백테스트하여 수익률이 가장 높은 최상의 숫자 조합을 스스로 찾아내는 로직.
3. **자동 성장(Auto-Update)**: 최적화된 결과물을 매주 일요일 밤 시스템이 스스로 자신의 코드에 덮어쓰고 커밋/푸시하여 평생 스스로 진화하도록 설정.

## 3. 다음 작업 단계 (Next Session)
1. `strategy_config.json` 생성 및 `StockAnalyzer` 클래스 생성자 수정.
2. 과거 성과 분석을 위한 수익률 측정 모듈 고도화.
3. `optimizer.py` 개발 및 `optimize.yml` 스케줄 등록.
